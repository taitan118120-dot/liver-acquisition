# Googleビジネスプロフィール 登録ガイド

## なぜ登録するのか
「ライバー事務所 石川」「ライバー事務所 金沢」で検索した時に、Google検索結果とGoogleマップに表示される。無料で放置型の集客導線。

---

## 登録手順

### ステップ1: Googleビジネスプロフィールにアクセス
1. https://business.google.com/ にアクセス
2. Googleアカウントでログイン

### ステップ2: ビジネス情報を入力

| 項目 | 入力内容 |
|------|----------|
| ビジネス名 | TAITAN PRO（タイタンプロ） |
| カテゴリ | 「タレント事務所」または「エンターテイメント事務所」 |
| 住所 | （石川県の事業所住所を入力） |
| サービス提供地域 | 日本全国（オンライン対応のため） |
| 電話番号 | （事業用電話番号） |
| Webサイト | https://taitan-pro-lp.netlify.app |

### ステップ3: 確認方法を選択
- ハガキ（住所確認）：2〜3週間
- 電話（場合により）：即日
- メール（場合により）：数日

### ステップ4: プロフィールを最適化

✅ 専属マネージャーが全力サポート
✅ 初期費用0円

まずは15分の無料オンライン面談から。お気軽にお問い合わせください。

▼ 詳しくはこちら
https://taitan-pro-lp.netlify.app

## 口コミ対策

### 口コミを集める方法
1. 所属ライバーに口コミ投稿を依頼
2. 面談を受けた方にも投稿を依頼（所属しなくてもOK）
3. 良い口コミには必ず返信する

### 口コミ返信テンプレート
```
口コミありがとうございます！
○○さんの配信がどんどん成長していて、こちらも嬉しいです。
これからも全力でサポートしますので、一緒に頑張りましょう！
```
