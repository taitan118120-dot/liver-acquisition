# 手動フォローガイド（1日20人）

## やり方

朝・昼・夜に分けてフォロー（1回7人くらい）

## Xで検索してフォローする人を探す

以下のキーワードで検索 → 「最新」タブ → 良さそうな人をフォロー

### 検索キーワード
```
配信初心者
Pococha 始めた
#初配信
#配信初心者
#ぽこちゃ
石川県 配信
金沢
ライバー 頑張る
配信 楽しかった
```

### フォローしていい人
- 配信始めたばかりの人
- Pocochaやってる人
- 石川県・北陸の人
- 普通の個人アカウント（顔写真ありなら尚良し）

### フォローしちゃダメな人
- プロフに「副業」「稼ぐ」「月収」がある → 業者
- 「事務所所属」「専属」がある → 引き抜きリスク
- フォロワー1万以上 → フォロバしてくれない
- 企業アカウント
- 投稿が全然ない人

## 1日のスケジュール

| 時間 | やること |
|------|---------|
| 朝 | Xで検索して7人フォロー（5分） |
| 昼 | 別のキーワードで7人フォロー（5分） |
| 夜 | 残り6人フォロー（5分） |
| 合計 | 20人 / 15分 |
