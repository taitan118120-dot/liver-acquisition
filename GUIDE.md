# TAITAN PRO 集客自動化 運用ガイド

## 3フェーズ戦略

### Phase 1: アカウント育成期（1〜2週間）
**目標: フォロワー100人**

やること（毎日）:
1. 価値提供ツイートを1日3〜4回投稿
2. ターゲット層のツイートに「いいね」30〜50回
3. 関連ツイートにリプライ 10〜15回
4. 関連アカウントをフォロー 20〜30回

コマンド:
```bash
# 手動エンゲージメント用のリスト（API不要、今すぐ使える）
python3 engager.py --manual

# 投稿テキスト確認
python3 post_scheduler.py --dry-run
```

ポイント:
- 朝・昼・夜に分けて作業する（一気にやると凍結リスク）
- 最初は売り込みツイートを控えて、価値提供を中心に
- 投稿コンテンツは posts/twitter_posts.json の phase: "growth" を使う

---

### Phase 2: DM開始期（2週間〜1ヶ月）
**目標: フォロワー300人 + DM送信開始**

やること:
1. Phase 1の継続
2. 募集ツイートを混ぜ始める（3回に1回くらい）
3. DMアウトリーチ開始（1日5〜10通）

コマンド:
```bash
# リード検索（API必要。なければ手動でリスト作成）
python3 run.py find --dry-run

# コピペ用DM生成（API不要）
python3 run.py dm --copy

# リード統計
python3 run.py stats
```

ポイント:
- DMは必ずプロフィールを見てからパーソナライズする
- 返信がなくても3日後にフォローアップDMを送る
- 反応が良いDMテンプレートを記録して改善する

---

### Phase 3: 自動化期（1ヶ月〜）
**目標: 月10件以上の面談獲得**

やること:
1. X API Basic ($100/月) を取得
2. 全自動化を稼働

コマンド:
```bash
# 自動投稿スケジュール開始
python3 run.py post --auto

# 自動いいね・リプライ・フォロー
python3 engager.py --all

# 自動DM送信
python3 run.py dm --platform twitter

# フォローアップ確認
python3 run.py followup

# ダッシュボード更新
python3 run.py dashboard
```

## 毎日のルーティン（Phase 1）

### 朝（10分）
1. `python3 engager.py --manual` でリスト確認
2. Xで検索していいね 15回 + リプライ 5回

### 昼（10分）
1. 投稿コンテンツを1つ投稿
2. いいね 15回 + リプライ 5回

### 夜（15分）
1. 投稿コンテンツを1〜2つ投稿
2. いいね 20回 + リプライ 5回
3. 関連アカウントをフォロー 20回

## Instagram運用

### 投稿頻度
- フィード投稿: 週3〜4回
- ストーリーズ: 毎日1〜2回
- リール: 週1〜2回（余裕があれば）

### ハッシュタグ戦略
投稿ごとに以下から10〜15個選んで使う:
- 大: #副業 #在宅ワーク #ライブ配信
- 中: #ライバー #ライバー募集 #配信者
- 小: #ライバーデビュー #TAITAN_PRO
